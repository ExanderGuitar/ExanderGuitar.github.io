Introducción a los contenidos básicos de HTML

Estructura básica

< !DOCTYPE html >

Siempre debe de ir al principio de todo documento html; además, indica el estándar de html que debe de usar el navegador para leer la página.

< html > < /html >

Marcan el inicio y el final del archivo que va a ser leído por el navegador. El resto de código ira entre estas marcas. Hay que fijarse la barra en la marca de cierre, es importante indicarlo.

< head > < /head >

Indican el encabezado de la página, usado normalmente para añadir links y metadatos.

< title > Título de la página < /title >

Se sitúa dentro del encabezamiento y sirve para mostrar el nombre de la página web en la barra superior del navegador o en las pestañas.

< body > < /body >

Aquí añadiremos toda la información que queremos que se muestre en la página web.

Elementos básicos

  • Encabezados

    • <h1> Titulo </h1> de forma general <hn> </hn> donde n=1…6
    Son usados como titulares, tienen un tamaño de fuente superior al cuerpo de texto normal pero posteriormente puede ser modificado.

  • Parágrafos

    • <p> cuerpo de texto </p>
    Se usan para añadir el cuerpo de texto, tiene un tamaño fijo.

  • Comentarios

    • <!-- comentario -->
    Ante la necesidad de aclarar una sección de código o para seccionarla.

  • Cambio de línea, espacios y otros elementos.

    • Para el cambio de línea se usa <br/>, no hace falta marca de salida.
    Para otros elementos

    • Espacio            &nbsp
    • Ampersand      &amp
    • Abrir marca     &lt
    • Cerrar marca    &gt

  • Listas

    • Como en la vida misma podemos ordenar la información listándola, puede ser de forma numerada (ordered list) o sin numerar (unordered list).

    • Numerada

      • <ol>
        <li> elemento 1 </li>
        <li> elemento 2 </li>
      </ol>

    • Sin numerar

      • <ul>
        <li> elemento 1 </li>
        <li> elemento 2 </li>
      </ul>

  • Enlaces

    • Son usados para redirigir a otra página web.
    <a href=“página web URL” target=“objetivo”> nombre del link o texto </a>

    La estructura contiene el elemento “a” para hacer referencia a lo que es, “href” sirve para indicar la URL, “target” se usa para indicar si queremos que la página se abra en una pestaña o en una ventana nueva y el nombre del link es un fragmento de texto que cuando se pinche en el se abrirá el enlace. Respecto a target, se puede omitir y entonces se abrirá en la misma ventana.

  • Imágenes

    • Podemos añadir imágenes de la siguiente forma. <img src=“URL o dirección de la imagen” alt”descripción de la imagen”/> El elemento alt es importante usarlo no solo por el texto explicativo que representa la imagen cuando pasamos el ratón por encima, sirve también para la accesibilidad de la página ya que ésta es leída por el sistema de voz.

  • Links

    • Útiles para añadir los metadatos al archivo html, indicándole donde cargarlos.
    <link href=“ruta o URL” type=“tipo” rel=“estilo”>

    Para añadir un archivo .css mediante link o usar una fuente externa.
    <link href=“URL” type=“text/css” rel=“stylesheet”>
