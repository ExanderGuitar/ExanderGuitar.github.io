Color

Éste parámetro sirve para cambiar el color de las letras o el color de fondo, que viene a ser un recuadro situado detrás.

  • Para cambiar el color de fuente

    • Tag {
      color: valor;
      }

  • Para cambiar el color del fondo

    • Tag {
      background-color: valor;
      }

El valor puede ser sustituido por:

  • Nombre del color, si el color esta formado por varias palabras tiene que estar entre comillas.

  • Formato rgb(255, 255, 255) donde cada valor representa el rojo, verde y azul respectivamente y sus valores numéricos van de 0…255, también puede usarse notación hexadecimal #AAFFAA; como funcionan en pares, si alguno de ellos se repite puede ponerse una sola letra. Por ejemplo: #AABBDD también puede ser #ABD

  • Formato hsl(360, 100%, 100%), significa Hue-Saturation-Lightness, donde el primer valor corresponde al color en una paleta de 360 grados, el segundo valor indica la saturación y el último el brillo.

  • Formato con alfa, puede usarse en los dos anteriores. El alfa, que es el último valor que varia entre 0 y 1, establece la transparencia.
    • rgba(255, 255, 255, 1)
    • hsla(360, 100%, 100%, 1)
