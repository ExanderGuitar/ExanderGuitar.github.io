Formularios en HTML

Formularios del inglés HTML Forms <form></form>

Los formularios en HTML se utilizan para recopilar los datos introducidos por el usuario y dirigirlos a una ubicación o a que se procesen.

La forma básica de declaración de un formulario es:

  • Étiqueta <form></form>
  • Atributo action -> determina dónde se manda la información
  • Atributo method -> se asigna un verbo HTTP. Por convención, el verbo se escribe en mayúsculas. Aunque no es necesario.
Entre medias de las etiquetas de apertura y cierre puede contener elementos hijos como títulos, párrafos, etc.

Entrada de texto del inglés Text Input <imput>

Se crea un campo de entrada utilizando la etiqueta <imput>. No tiene etiqueta de cierre. Puede contener en su interior diferentes atributos.

  • Atributo type -> determina como se renderiza y el tipo de datos que acepta el campo de entrada.
  • Atributo name -> necesario para identificar la entrada con un nombre.
  • Atributo value -> se asigna cuando se escribe en el campo de entrada. Si inicialmente se le asigna un valor se mostrará como texto preasignado.
Al mandarse este formulario, se produce una unión entre el valor del atributo “name” y el valor del atributo “value”.

Etiqueta del inglés Label <label></label>

La etiqueta sirve para asociar una descripción a un campo de entrada. Para asociarse, el campo de entrada necesita un atributo “id” que lo identifique. La etiqueta relaciona el campo de entrada mediante su atributo “for” acompañado del mismo valor dado para el atributo “id”.

Contraseña del inglés Type Password

El atributo “Type” puede aceptar como valor password. Al escribir en el campo de entrada de texto se ocultara mediante otros caracteres el texto introducido. Útil para introducir contraseñas.

Número del inglés Type Number

El valor number para el atributo “Type” sirve para aceptar como entrada solamente números. Si se añade el atributo “step”, junto al campo de entrada se representan unas flechas de subida y bajada. Cuando se pulsan, cambiaran el texto en función del valor asignado al atributo “step”.

Rango del inglés Type Range

Este valor range genera un rango con un mínimo min, un máximo max y un paso determinado por step.

Marca de selección del inglés CheckBox

Se utiliza para la selección de varias opciones. Se usa el mismo valor para el atributo “name” pero diferente en el “id”. Necesita del atributo “value” para mostrar el texto.

Botones circulares del inglés Radio Buttons

Da a elegir una de entre varías respuestas. Se necesita de una etiqueta para mostrar el valor. Si se quiere agrupar varios botones circulares se usa el mismo valor en “names”.

Lista desplegable del ingles Dropdown List <select></select>

Da a elegir entre una gran cantidad de opciones en una lista. Se usa el elemento <option></option> para generar cada entrada y se incluye el atributo “value”.

Entrada de lista de datos del ingles Datalist Input <datalist></datalist>

Tiene parecido con una lista desplegable pero se permite la entrada por teclado para ir cercando la búsqueda. Debe ir acompañado del tipo text y el atributo “list” que hace referencia al “id” de la lista de datos. Si que muestran su valor cada elemento. Cada elemento de la lista de datos debe de incluirse con la etiqueta

Área de texto del ingles Textarea <textarea></textarea>

Permite la introducción de una mayor cantidad de texto. Se añaden filas “rows” y columnas “cols”. Entre las etiquetas se puede identificar texto por defecto para mostrar.

Botón de subida del ingles Submit Button

Se utiliza el valor submit para el atributo “tope” para generar un botón. Este se utiliza para mandar el formulario. El atributo “value” muestra el texto en el botón.
