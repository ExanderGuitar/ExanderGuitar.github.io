Estructura básica CSS

En un archivo CSS no tenemos una estructura rígida y definida como en un archivo HTML. Los archivos CSS se usan para modificar el estilo de la página web o de algunos elementos. Por lo tanto lo que hacemos es modificar éstos individualmente mediante código. La estructura básica en CSS es la siguiente:

tag {
  parámetro: valor;
  }

Tag hace referencia al mismo que en el archivo HTML queremos modificar, puede ser un titular o un parágrafo, etc.
El parámetro es que queremos modificar, hay bastantes de ellos como el color, la fuente, el tamaño…
El valor es la nueva información que recibe el parámetro en cuestión siendo el nombre del color o el tamaño elegido para la fuente, etc.
Si queremos que los cambios afecten a la vez a varios elementos podemos encadenarlos con una coma, por ejemplo: h1, h2, h3. Por otra parte si queremos que el cambio afecte a todos los elementos usaremos un * como tag.
Por último, para realizar comentarios no es como en HTML, aquí usaremos la siguiente estructura.
/* comentario */
